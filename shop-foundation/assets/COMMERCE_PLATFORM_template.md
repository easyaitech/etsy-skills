- 商品字段与上架信息：`listing-catalog/references/platforms/xiaohongshu.md`
- 订单 / 客服 / 售后字段：`orders-customers/references/platforms/xiaohongshu.md`
- 图文 / 视频发布任务：`publish-composer/references/platform-publishing-model.md`
- 自动发布状态：`social-publisher/references/adapter-registry.md` 当前标记为 planned/manual-only；只能建草稿和人工对账，不能自动上传或发布。
- 商品结构：SPU / SPL / SPV / ITEM。
- 关键字段：品牌 ID、末级类目 ID、商品名称、商品简称、销售规格、SKU 规格、商品图、描述、商品特色、产品参数、FAQ、使用指南图、图文详情图、条码、重量、原产国、材质/用途/厂家、海关备案图、价格、库存、上下架状态、审核状态。
- 商品图规则：商品图片支持 800x800 或 750x1000，JPG / PNG / JPEG，比例 1:1 或 3:4；使用指南图和图文详情图按小红书 preset 的尺寸限制。
- 订单字段：订单 ID、包裹 ID、包裹状态、售后状态、取消状态、SKU 明细、承诺最晚发货时间、物流模式、拆单发货、收件地区、openAddressId。
- 自动化边界：默认只准备字段、文案和核对清单，不直接调用小红书后台或 Open API 创建商品、上架、提交审核、发布笔记、发货、改快递单号、退款或售后审核。
