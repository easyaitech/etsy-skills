1. 从现有素材切出 15 个片段（5H / 7B / 3C）
2. 填写下方表格
3. 手动试组合 5 套（见第二张表），记录观感

### 填写指南

- **文件名**：切好的片段文件名，如 `hook-candle-warm-01.mp4`
- **stage**：hook / body / close
- **产品**：对应的商品 / SKU（至少覆盖 1-2 个产品）
- **时长**：秒数（Hook 建议 2-5s，Body 5-15s，Close 2-5s）
- **风格标签**：极简 / 暖调 / 冷调 / 动感 / 安静 / 手工感 / 自然光（可多选）
- **mood**：energetic / calm / cozy / playful / elegant（单选）
- **内容简述**：一句话说这段拍了什么

- **5** — 完全自然，像一条完整视频

## 切片建议

从现有视频素材切片时的实操 tips：

- **Hook 切法**：找视频前 3-5 秒最抓眼球的画面（产品特写、手部动作开始瞬间、光影变化）
- **Body 切法**：展示产品使用/制作过程的中段，5-15 秒一个完整动作
- **Close 切法**：产品全貌 + 品牌 logo 露出，或手部放下产品的动作，2-5 秒
- **剪切点**：尽量在动作完成或画面静止时切，不要切在运镜中间
- **分辨率**：保持竖屏 9:16 或接近比例，FFmpeg 会 pad 但原始越接近越好
