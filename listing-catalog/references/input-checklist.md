# 写 listing 前的输入清单

进入模式 B（写 listing 文案）前按本清单盘点用户已提供 / 还缺的输入。

## 必填

| 项目 | 说明 | 用途 |
|---|---|---|
| 产品图（≥1 张）| 用 vision 工具看图 | 判断材质质感、釉色、形态、工艺细节——直接进 description |
| 中文产品名 / 类目 | 例："建盏 / 兔毫盏" | 起英文 title 的语义起点 |
| 关键参数 | 材质、尺寸/容量、重量、颜色、工艺（手作/拉胚/手绘/柴烧 等）| 进 description 的 specs 段 + tags 词源 |

缺任一必填项 → 停下来一次性问全，不要边写边追问。

## 强烈建议但可选

- 创作灵感 / 故事（独立工艺品的 description 头段很吃这个）
- 使用场景 / 受众（"一人独饮"、"送礼"）— 影响文案调性与 tags
- 同类竞品 listing 链接（参考 SEO 词，不是抄文案）

可选项缺失**不必停下来**——直接基于产品图推测着写，但在草稿里标注「这部分基于产品图推测，你看要不要补真实背景」。

## 不要向用户索要

- 英文 title / description / tags / materials（这是 skill 的产出）
- 政策段（从 SHOP.md 自动取）
